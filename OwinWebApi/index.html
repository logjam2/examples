LogJam Example - OWIN & Web-API

  • LogJam Tracing in Web API request
    (using LogJam.Trace.Tracer)
  • Web API Tracing in Web API request
    (using System.Web.Http.Tracing.ITraceWriter forwarding to LogJam.Trace.Tracer)
  • Exception thrown from Web API action method
    (uses LogJam.WebApi.LogJamExceptionLogger to log Web API exceptions to LogJam.Trace.Tracer)
  • Web API request that waits the specified amount of time before returning
  • OWIN Tracing in OWIN request
    (using Microsoft.Owin.Logging.ILogger forwarding to LogJam.Trace.Tracer)
  • Exception thrown from OWIN request
    (uses Microsoft.Owin.Diagnostics.ErrorPageMiddleware or LogJam.Owin.ExceptionLoggingMiddleware to log unhandled OWIN exceptions to LogJam.Trace.Tracer)
